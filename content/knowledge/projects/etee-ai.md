# Etee.ai

- Project type: Personal project
- Timeline: 2023
- Role: Technical Partner

## Problem

Print-on-demand e-commerce usually requires fragmented design, preview, production, and fulfillment workflows, which creates friction for end users and merchants.

## Solution

Bao helped develop an e-commerce application for custom t-shirt sales with AI-powered image generation. Users can create designs with AI, apply them directly to shirt templates in real time, and send finalized designs automatically to printing partners for production and delivery.

## Contributions

- Worked on the technical direction of an AI-assisted e-commerce platform.
- Helped connect image generation, live template application, and downstream production handoff.
- Contributed to a product that combines AI features with real operational workflow.

## Technical stack

- Node.js
- AI image generation
- E-commerce workflow integration

## Impact

- Shows Bao's interest in practical AI products rather than AI features in isolation.
- Highlights experience building systems that need both user-facing value and operational execution.
